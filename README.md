# Bavir Interiors | Luxury Interior Design Studio

Transforming spaces into timeless interiors. Premium residential and commercial interior design services in India.

## Project Overview

This is the primary repository for the Bavir Interiors official website. It features a modern, responsive design built with high-end aesthetics and optimized performance.

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS & Vanilla CSS
- **Components**: shadcn/ui & Framer Motion
- **Icons**: Lucide React
- **Data Management**: React Query

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/SujitD56/bavir-interiors-design-hub-1b54c6ca.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Key Features

- **Interactive Portfolio Gallery**: Modular lightbox for viewing design variations.
- **Service Calculators**: Cost estimators for modular kitchens and other services.
- **Unified Services Data**: Centralized management of service details and pricing.
- **Premium Animations**: Smooth transitions and hover effects using Framer Motion.

## Deployment

Pushed changes to the `main` branch are automatically built and deployed.
